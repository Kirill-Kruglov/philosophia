## Accepted as blocking

1. **C7 missing (Opus; sharpened by GPT).** No positive control for TRUE
   independent convergence: an instrument that calls every convergence
   "glue" passes K2+C1 while blind to real world-invariants. C7 must fix
   three facts at once: both paths competent; same externally-known
   invariant; implementations separated by a REGISTERED provenance
   protocol (separate code paths, RNGs, authors; allowed common causes =
   W and I only; explicit causal graph of shared components). Without the
   protocol it is one more pair with assigned independence.
   *Empirical seed already exists:* scout 09's (G,P) on honest cycles —
   same correct destination n, cleared as clean (excess 0.05) — true
   convergence was already readable; it was never registered as a control
   and carries no provenance protocol. Opus's dichotomy ("C7 impossible ⇒
   wall of the world") is answered constructively: C7 is buildable.

2. **Three-variable conflation (GPT §2).** Destination agreement (A) ≠
   path dependence (B) ≠ world truth (C). K1 measures A; v3 measures B;
   "world-credit" rhetoric smuggles C. Verdicts split into two axes:
   `DESTINATION_SAME/DIFFERENT` × `RESIDUALLY_DEPENDENT /
   CLEAN_UNDER_CHANNELS / UNKNOWN`; world-credit only as a separate rule
   over both axes plus a competence gate.

3. **"Cayley glue = dependence" was an assigned label, not a fact.** The
   theorem links targets, not procedures; in Z/nZ agreement of competent
   order-finders is world-forced. The 0.093/0.094 clone≈Cayley tie has a
   second live interpretation: the metric may measure shared
   interface/world forcing, not shared construction. Empirically resolved
   by TWO A/G pairs:
   - **A/G-translated** — G built/parameterized through A: the registered
     dependent sham;
   - **A/G-independent-build** — separate implementations under the C7
     provenance protocol.
   The experiment's question becomes: does v3 distinguish the two
   genealogies of the same theorem-glued destination? (This is the line's
   founding question — same wall, different paths — made measurable.)

4. **K1 downgraded.** The naive-detector fall on a dependent pair follows
   from definitions; the run checks the harness. K1 stays as a regression
   arm, is not citable epistemology.

5. **Statistics (GPT §5).** Reference field: 6–8 independently built
   mechanisms; leave-both-candidate-members-out spelled out; reference
   pairs grouped by construction, not counted as independent edges;
   leave-one-reference-out contamination analysis; behavior under one
   poisoned reference preregistered. Operating characteristics BEFORE
   lock: FPR/FNR/power (incl. on C7 and masked-postulate), MCSE,
   sensitivity to worlds/seeds/references, multiple-comparison policy.
   **Dev-set contamination:** ten scouts made this world family a
   development set; the locked run on it is internal validation only. A
   holdout family (new worlds + new shams, generated/disclosed only after
   lock, ideally by an agent that has not seen metric internals) is
   required for any confirmation claim.

6. **Dual-implementation ground truth (GPT, replacing Opus's crack 3).**
   Two oracles — anonymous permutation/transition table AND modular
   `net` arithmetic — cross-checked exhaustively on small n; evaluator
   uses the hidden one; languages import no evaluator helpers. The
   truth-language asymmetry is an engineering/provenance risk, handled by
   construction, not an ontological hole in this fixed world.

7. **K3 identifiability (GPT).** Answerable but must be written out: the
   traces DIFFER (languages consume oracle answers; the inconsistent
   oracle yields a different answer stream, destinations do not
   stabilize). The prereg must state the exact input signature of the
   naive detector and the observable that differs (the o_i stream).

## Core amendments (they correct the LINE's core, not just the prereg)

- **π example is wrong** as algorithmic randomness: π is computable,
  K(first n digits) = O(log n), not Martin-Löf random. Survives only as
  "passes a bounded battery of statistical tests". The dialogue's chain
  "language-blind = incompressible = machine change makes compressible"
  loses this example.
- **Invariance theorem was over-applied.** The scout-08 recursion
  (disjointness not certifiable from inside) stands on its own three
  measurements; it is NOT a consequence of Kolmogorov's invariance
  theorem. Analogy, not derivation.
- **"Insight trigger uncomputable by necessity" narrowed** to: no known
  universal complete computable trigger for arbitrary domains; bounded
  triggers within a fixed meta-language are computable (DreamCoder,
  AlphaGeometry are counterexamples to the strong form).
- **"Reality = invariance under description change" downgraded** from an
  identity to Wimsatt-robustness: invariance under a preregistered,
  diverse intervention family is evidence of robustness — conditional on
  path competence and bounded error dependence. This is the operational
  form the instrument actually uses.
- **Current honest label for v3 (GPT, verbatim):** a candidate detector
  of residual path dependence under fixed world, interface, perturbation
  family, and declared reference constructions. Not yet a detector of
  "the same wall"; not yet an engineering mechanism of experience.
