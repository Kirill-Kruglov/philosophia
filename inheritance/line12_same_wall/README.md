# Line 12 inheritance: The Same Wall

This directory is a verbatim extraction of the immediate experimental ancestor
of Philosophia.

- `LINE12_MAP.md` records the complete route and its interpretation.
- `experiment_A/decision.json` is the harness-valid primary decision.
- `experiment_A/holdout_result.json` records the escrowed holdout that confirmed
  the token core and broke the full blade's portability.
- `non_citable/scouts/` preserves exploratory work without promoting it to
  evidence.

The old preregistration lock belongs to Ascesis history. It is retained for
provenance only and cannot authorize a new run in this repository.
